# Вероятностный генератор

### Генерирует какойто из вариантов в соответствии с вероятностью выпадения каждого варианта.

Как пользоваться:
- Создать объект класса Probability_generator
- С помощью функции AddVariant добавить возможные выпадающие варианты, аргумент "probability" отвечает за вес каждого из вариантов (чем он больше - тем чаще он будет выпадать), проще всего сделать так, чтобы итоговая сумма весов была равна 100, тогда каждый вес примет вероятность своего выпадения.
- Получать какой то из вариантов с помощью функции Next. Она вернет object добавленного варианта из второго пункта.
- С помощью функции Clear можно очистить полностью список генератора
